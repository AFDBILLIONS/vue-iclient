# 组件基础类型

## CircleStyle 

点样式

| 参数   | 说明                        | 类型                                                                       | 可选值 | 默认值 |
| :----- | :-------------------------- | :------------------------------------------------------------------------- | :----- | :----- |
| paint  | MapboxGL 点图层 Paint 配置  | [pain](https://docs.mapbox.com/mapbox-gl-js/style-spec/#paint-property)    | -      | -      |
| layout | MapboxGL 点图层 Layout 配置 | [layout](https://docs.mapbox.com/mapbox-gl-js/style-spec/#layout-property) | -      | -      |
